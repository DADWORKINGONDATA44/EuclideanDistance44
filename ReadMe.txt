# Avatar Assistant Website

An interactive Next.js application for creating and customizing AI-powered avatar assistants.

## 🌐 Live Preview
Once deployed with Vercel, your site will be live at:
```
https://your-project-name.vercel.app
```

## 🚀 Features
- Video background showcasing avatars
- Explore features (Read Emails, AI Research, Create Website)
- Animated hero section
- Stub pages for each functionality

## 🛠️ Getting Started

### 1. Clone the Repo
```bash
git clone https://github.com/yourusername/avatar-assistant.git
cd avatar-assistant
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Run the App Locally
```bash
npm run dev
```
Then open `http://localhost:3000` in your browser.

## 📦 Deploy with Vercel
1. Push this project to GitHub
2. Go to [https://vercel.com](https://vercel.com)
3. Import the repo and follow the deploy steps
